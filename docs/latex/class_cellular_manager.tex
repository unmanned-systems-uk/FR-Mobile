\doxysubsection{Detailed Description}
Cellular Manager implementing comprehensive network communication. 

This class provides a complete interface for cellular communication using the SIM7600X module. Features include\+:


\begin{DoxyItemize}
\item Network connection management with automatic retry
\item HTTP/\+HTTPS operations for data upload
\item Network time synchronization (NITZ)
\item Signal quality monitoring and reporting
\item SIM card management and authentication
\item Data transmission in configurable chunks
\item Connection health monitoring and recovery
\item Comprehensive error handling and logging
\item Thread-\/safe operations with proper synchronization 
\end{DoxyItemize}

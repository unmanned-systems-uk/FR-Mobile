\doxysubsection{Detailed Description}
Wi\+Fi Scanner class for capturing probe requests in promiscuous mode. 

This class implements the \doxylink{class_i_scanner}{IScanner} interface to provide Wi\+Fi scanning functionality. It captures probe requests from nearby devices and stores them for analysis or transmission.

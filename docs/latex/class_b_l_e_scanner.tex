\doxysubsection{Detailed Description}
BLE Scanner class for capturing and processing BLE advertisements. 

This class implements the \doxylink{class_i_scanner}{IScanner} interface to provide comprehensive BLE scanning functionality. It captures BLE advertisements, processes device information, and integrates with the logging and storage systems.

Key features\+:
\begin{DoxyItemize}
\item Thread-\/safe scanning operations
\item Configurable scan parameters (interval, window, RSSI threshold)
\item Real-\/time result callbacks
\item Automatic data persistence
\item Platform abstraction (ESP32/development)
\item Comprehensive error handling and logging 
\end{DoxyItemize}

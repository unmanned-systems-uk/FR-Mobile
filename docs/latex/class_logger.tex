\doxysubsection{Detailed Description}
Comprehensive logging system for the forestry research device. 

Features\+:
\begin{DoxyItemize}
\item Multiple log levels (DEBUG, INFO, WARNING, ERROR, CRITICAL)
\item Dual output (console + SD card)
\item Automatic file rotation by size and time
\item Thread-\/safe operation
\item Asynchronous logging to prevent blocking
\item Configurable formatting
\item Component-\/based filtering 
\end{DoxyItemize}

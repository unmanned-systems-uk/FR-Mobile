\doxysubsection{Detailed Description}
Power Manager implementing comprehensive power control and optimization. 

This class provides complete power management for the ESP32-\/based forestry research device. Features include\+:


\begin{DoxyItemize}
\item Deep sleep modes with configurable wakeup sources
\item Peripheral power control and optimization
\item Battery-\/aware power management with emergency procedures
\item Watchdog timer configuration and monitoring
\item Wake/sleep cycle statistics and analysis
\item GPIO hold states for deep sleep persistence
\item Reset and wakeup cause analysis
\item Power consumption monitoring and optimization 
\end{DoxyItemize}

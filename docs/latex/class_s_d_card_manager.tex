\doxysubsection{Detailed Description}
SD Card Manager implementing comprehensive file operations and data management. 

This class provides a complete interface for SD card operations optimized for the forestry research device. Features include\+:


\begin{DoxyItemize}
\item Reliable file operations with error recovery
\item CSV data logging with proper formatting
\item Capacity monitoring and low-\/space warnings
\item Directory management and file organization
\item Atomic write operations for data integrity
\item Performance monitoring and statistics
\item Thread-\/safe operations with proper locking
\item Automatic mount/unmount handling 
\end{DoxyItemize}

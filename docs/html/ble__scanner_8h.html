Detailed Description

BLE Scanner class for capturing Bluetooth Low Energy advertisements.

Author
Anthony Kirk
Date
2025-07-29
Version
1.0.0

This class implements the IScanner interface to provide BLE scanning functionality. It captures BLE advertisements from nearby devices and stores them for analysis or transmission. Designed primarily for ESP32 but includes development platform support.

Detailed Description

BLE Scanner class for capturing and processing BLE advertisements.

This class implements the IScanner interface to provide comprehensive BLE scanning functionality. It captures BLE advertisements, processes device information, and integrates with the logging and storage systems.

Key features:

  • Thread-safe scanning operations
  • Configurable scan parameters (interval, window, RSSI threshold)
  • Real-time result callbacks
  • Automatic data persistence
  • Platform abstraction (ESP32/development)
  • Comprehensive error handling and logging
